import { PolicySection, ReturnButton } from '@/components/mdx'

<PolicySection title="配送方法">
限定グッズの世界配送を承っております。すべてのご注文は、安全な配送を確保するため信頼できる配送業者を通じて発送いたします。

- 国際標準配送（7-14営業日）
- 国際速達配送（3-7営業日）
- 国内配送（2-5営業日）
</PolicySection>

<PolicySection title="配送料金">
すべてのご注文に対して**送料無料**でお届けいたします。最低注文金額の設定はございません。

- 世界中どこでも標準配送無料
- ¥10,000以上のご注文で速達配送無料
- ほとんどの国で追加の関税はかかりません
</PolicySection>

<PolicySection title="処理時間">
ご注文は通常1-3営業日以内に処理されます。繁忙期や特別リリース時には、処理に最大5営業日かかる場合があります。商品が発送されましたら、確認メールをお送りいたします。
</PolicySection>

<PolicySection title="配送期間">
**国内配送**
- 標準：2-5営業日
- 速達：1-3営業日

**アジア太平洋地域**
- 標準：5-10営業日
- 速達：3-7営業日

**北米・ヨーロッパ**
- 標準：7-14営業日
- 速達：5-10営業日

**その他の地域**
- 標準：10-21営業日
- 速達：7-14営業日
</PolicySection>

<PolicySection title="配送追跡">
商品が発送されましたら、追跡番号をメールでお送りいたします。この番号を使用して、配送パートナーのウェブサイトで荷物を追跡できます。
</PolicySection>

<PolicySection title="配送制限">
世界中のほとんどの国に配送いたしますが、一部制限が適用される場合があります：

- 遠隔地や農村部では配送時間が延長される場合があります
- 一部の国では特定の素材に輸入制限がある場合があります
- 税関での遅延が発生する場合があり、これは当社の管理外です
- 国際配送では私書箱住所が受け付けられない場合があります
</PolicySection>

<PolicySection title="破損・紛失した荷物">
荷物が破損して到着したり、輸送中に紛失した場合は、すぐにinfo@frankfurtlin-store.comまでご連絡ください。配送パートナーと協力して問題を解決し、確実に商品をお届けいたします。
</PolicySection>

<PolicySection title="お問い合わせ">
